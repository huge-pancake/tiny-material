With icon, 带有图标的

Way 1, use slot and custom source (recommended), 第一种方式, 使用 slot 和自定义的源 (推荐)

This way can reduce network spending. 该方法可减少网络开销

Also, you have more customize on this way, like use another icon set (in iconify), or use image as a icon.

同时, 你可以拥有更多的自定义空间, 例如使用另一个图标集 (在 iconify 内), 或是拿图片当作图标使用.

Way 2, use attribute, 第二种方法, 使用属性

Sometimes, maybe you want to use button component with a <input>, but then you will find that this is not feasible. So we provide function to custom inner element. See the following example(s).

有时, 也许你希望将 <input> 的功能同 button 组件的样式一同使用, 但你将会发现这不可行. 所以我们提供自定义内部元素的功能. 见下例子.

Accessibility, 可访问性

Use data-aria-label attribute to custom aria-label.

使用 data-aria-label 属性来自定义 aria-label.
