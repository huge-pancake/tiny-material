Custom inner element, 自定义内部元素

Sometimes, maybe you want to use FAB component with a <input>, but then you will find that this is not feasible. So we provide function to custom inner element. See the following example(s).

有时, 也许你希望将 <input> 的功能同 FAB 组件的样式一同使用, 但你将会发现这不可行. 所以我们提供自定义内部元素的功能. 见下例子.

Link, 链接

<a> must with tabindex="-1" style="text-decoration: none;"

<a> 必须有 tabindex="-1" style="text-decoration: none;"

Accessibility, 可访问性

Use data-aria-label attribute to custom aria-label.

使用 data-aria-label 属性来自定义 aria-label.

Support all the other ARIA attributes for fab, see details in source code.

其他用于按钮的 ARIA 属性全部支持, 详见源代码.
