Attributes, 属性

Types:

  • single choice means choose one of all names
  • toggle means give a attribute with no content for true, do not give this attribute for false
Name, 名称 Type, 类型 Default value, 默认值 Description, 描述
(small) / medium / large single choice (small) Controls top app bar's style
centered toggle false Controls top app bar's title align is left or is center, only valid when the style is small
static toggle false Controls top app bar's position is or is not static, always for demo
no-nav-btn toggle false Controls top app bar has or has no navigation button container
no-contents toggle false Controls top app bar has or has no contents container, always for demo
